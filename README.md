# 🎯 AI Interview Preparation System

## 🚀 Project Overview
This project is an AI-powered Interview Preparation System that helps users practice technical interviews. It generates skill-based questions, evaluates user answers, calculates scores, and provides feedback in an interactive interface.

## 💡 Features
- 🧠 Skill-based question generation (Python, SQL)
- 🎯 Real-time answer evaluation
- 📊 Score tracking system
- ⏱ Time tracking for each answer
- 📜 Answer review after completion
- 📥 Download interview report
- ⭐ Clean UI with interactive elements
- 💾 SQLite database integration

## 🛠️ Tech Stack
- Python
- Streamlit
- SQLite
- Basic AI logic (keyword-based evaluation)
- HTML

## ▶️ How to Run
1. Clone the repository
2. Go to project folder
3. Install dependencies:
   pip install -r requirements.txt
4. Run the app:
   streamlit run app.py
5. Open in browser:
   http://localhost:8501

## 🧪 How to Use
1. Select skills (Python / SQL)
2. Click "Start Interview"
3. Answer questions
4. Get feedback and score
5. Download report
